# IEEE Student Branch Data Observatory

## Project Overview

This project analyzes IEEE Student Branch event data using Python, Pandas, and Matplotlib. The project uses a dataset of 30 IEEE events to study participation trends, event popularity, member retention, and budget utilization.

## Objectives

- Analyze event attendance and registrations.
- Compare registrations with actual attendance.
- Identify the most popular event type.
- Analyze member retention.
- Study budget allocation.
- Build a simple dashboard using Python.

## Technologies Used

- `retention.png` – Member retention graph
- `budget.png` – Budget analysis graph
- `final_report.pdf` – Project report

## Key Findings

- Hackathons recorded the highest average attendance.
- Workshops required the highest average budget.
- Returning members were higher than new members.
- Registrations were generally higher than attendance.
